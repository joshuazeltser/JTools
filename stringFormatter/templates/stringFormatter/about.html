About JTools

Who am I?

My name is Josh Zeltser and I am a Computing BEng graduate from Imperial College London. I am a passionate programmer from London always looking to solve the everyday problems that we are faced with using the tools at our desposal. I am also very keen to try out new technologies when carrying out new projects. JTools has given me the chance to both try out new technologies while solving some everyday challenges that at least I face on a daily basis.

What is JTools?

JTools brings together a selection of custom made tools into one easy to use web-app for everyday use. JTools aims to help users with the common everyday challenges of computer use, saving time and making sure that they complete their tasks as efficiently as possible. A lot of the tasks that are solved by JTools are problems that I myself have faced on a daily basis as I go about my everyday life. I often go online looking for tools to solve these problems, but usually end up using multiple websites or having to search for a long time to find a good one. JTools has been created to remove the hassle of having to use these websites by providing one site with many tools. Tools are being constantly added to help solve the many problems out there.

What technologies were used to create JTools?

This project was created using the Django framework ( Django ) and the backend has therefore been completely written using Python. The reason I decided to use Django for this project is that it allows me to easily create this project using a Model View Controller architecture which is set up for me. This makes the code very easy to understand and for it to be editted/added to in the future. For the front-end I used the very simple to use Bootsrap framework ( Bootstrap ) creating the design that you see today. All of the code used for this project can be found at Github Repository .
